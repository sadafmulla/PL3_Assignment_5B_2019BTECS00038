Program to converts binary number to decimal number.

Code

            #include 
            
                int main()
                {
                    int binnum, decnum=0, i=1, rem;
                    printf("Enter any Binary Number: " );
                    scanf ( "%d",&binnum );
                    while(binnum!=0)
                    {
                        rem = binnum%10;
                        decnum = decnum + (rem*i);
                        i = i*2;
                        binnum = binnum/10;
                    }
                    printf("\nEquivalent Decimal Value = %d",decnum);
                    return 0;
                }

Sample input

Enter any Binary Number: 100

Sample output

Enter any Binary Number: 100

Variable used in the program

binnum, decnum, i, , rem
